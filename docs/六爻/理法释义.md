第七章 六冲卦/地支相冲

# 1 月破



## 1.1 月破含义

不论所遇何事，快快走开

不能停留，冲之即动

【化学性质】不稳定，马上发生变化

【逻辑名词】互斥，排斥



:::tip

测忧患，测疾病，速速退去
测生产，六冲卦，顺利生产，与之相对应的，合住，难产
测怀孕，六冲卦，保不住，容易流产

:::

月破，而安静，实破的年月日发生的，不论过去未来

月破，且发动，下个月出月不破，就会发挥作用。也有实破而应验

用神生世，却月破，那就是无意生世，等同于无情。



## 1.2 静爻月破

写成：`静爻 月破`，或者`* 安静 月破`

月破之爻，

- 力量大幅削弱，残废



日月冲破，特殊情况

- 有时同上述

- 有时候反而没有很大问题，日月双冲反为吉（因物极必反）



## 1.3 发动月破

写成：`动爻 月破`，或`* 发动 月破`



## 1.4 动化月破

> 日冲变爻

写成：`动爻 化月破`，或`* 发动 化月破`

用神化回头克，回头克月破了，

- 变爻主未来之事

- 可以应下个月出月不破

## 1.5 飞伏月破



### 1.5.1 飞神月破

写成：`财|官|父|兄|孙 飞神 月破 `

如果非飞生伏，对于伏神来说，吉兆。

如果是飞生伏，对于伏神来说，降级。



:::tip

久病之人测病，不宜元神逢冲，元神逢冲则死
如：用神伏藏，飞生伏神，不宜元神月破。

:::


- 也没有其他动爻补充，着重点落在父母伏藏，飞生伏，元神月破的问题上。

  - 申月占，合破亥月死。

  - 他月占，元神逢冲申月死。再有一个就是元神入墓的未月死

最终在亥月去世。

# 2 日冲

日冲没有明显的吉凶，只是提示这个位置的人物/事件（近期）活跃，有些动作。

日冲静爻，暗中活跃。

日冲动爻，冲之愈急。特别强调近期的概念。



## 2.1 暗动

> 日冲静爻为暗动。

### 2.1.1

暗动之爻，能够**生克他爻，<font color="red">动爻越少，吉凶导向越明显</font>**。土爻暗动能够**入墓他爻**。

- 如果暗动来克，代表暗中不利。

- 如果暗动来生，代表有人帮忙了。



```
占题：客户能够签单/能否有财进
丁酉年 癸丑月 壬寅日 丁未时（空亡:辰巳）
天火同人 (离宫,归魂)      泽火革 (坎宫)
白虎  子孙戌土○ 应 子孙未土″
腾蛇  妻财申金′    
勾陈  兄弟午火′    
朱雀  官鬼亥水′ 世 
青龙  子孙丑土″    
玄武  父母卯木′    
```

分析：

- 约以父母为用神，求财多少才是看妻财。
- 应爻独发合了父母，对方为的是这个项目来的。
- 应爻化退，单看这一点好像是反悔了。但是退神被月建冲了，退无可退，不再化退了，反而不是反悔。
- 应爻发动克世，克为拒绝。但是间爻妻财暗动生世，引化了克世的力，构成连续相生，吉兆。间爻暗动生世，中间人帮自己说话了。

结果卦主与客户谈了半天没啥进展,有个朋友在旁边一直帮忙说话,最终签约成功。卦中的申金就是指这个朋友。

```

天泽履 (艮宫)      天水讼 (离宫,游魂)

勾陈          官鬼卯木′ 应 
朱雀          父母巳火○    官鬼寅木″
```

分析：

- 取象上看，子孙在五爻为路，侄子外出在路上，父母独发来克子孙，父母为车初爻为脚，车轮，不利孙辈，独发化出官鬼来冲世爻子孙，有冲撞之象。
- 侄子的人身安全，以子孙为用神，子孙月扶，日生了问题不大，又有兄弟暗动来生，吉兆。
- 元神兄弟戌土临下个月地支，暗动来生，下个月肯定恢复过来了

后来反馈，10月12日开始恢复知觉,可以认人了，时间正是戌月。

反馈：母亲最近在楼梯上摔倒了,伤了头部,肠胃长期不好。父亲的腰部也不好,以前受过伤,过了大半年左右，父亲又出了个小车祸,赔了不少钱。

### 2.1.2 暗动的力量大小

要知道暗动之爻力量大小？看月建是否生扶。

- 如果月建生扶，特别是用神月扶，这个暗动的**生克相当有力**。

- 如果月不生扶，暗动了也兴不起波澜。只是说近期有这样一件小事。
