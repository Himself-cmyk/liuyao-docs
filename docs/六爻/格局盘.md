### 官鬼持世

官鬼代表以下人物：

- 老师一类的学校职工，
- 代表领导以及自己的工作，
- 代表城管一类的政府方面的人员，
- 代表盗贼，犯人，绑架犯一类，
- 代表死者，仙家，神佛一类，
- 代表各种场合的工作人员，
- （女占）代表丈夫。

官鬼同时也代表不安。我们作为被管理的一方，自然不希望他们有什么动作。所谓“诸占不宜鬼动”。官鬼独发，官鬼暗动，官鬼持世，日月皆鬼，都在一定程度上，强化了这个卦中官鬼的信息比重。

如果占问车间生产，官鬼独发，可以代表灵异作祟，发生事故。

如果占问老师是否抽查到自己背诵，官鬼持世，官鬼暗动，很有可能。

分析：

- 偏向于离开工作的，请假，不像上班，喜子孙
- 偏向参与工作的，加班，工作量增加，偏重官鬼
- 这种问题类似于会不会被老师点名，会不会被领导找麻烦。官鬼持世，官鬼暗动会。

### 元神独发

这时看用神的旺衰。

:::tip

男测感情为例，

忌神兄弟持世，妻财伏藏，女人离开的状态，离线，找不到人。

妻财月破，子孙独发化鬼，

:::

```
占题：是否该见女友最后一面
丁酉年 壬子月 庚午日 甲申时（空亡:戌亥）
水火既济 (坎宫)      水山蹇 (兑宫)
腾蛇          兄弟子水″ 应 
勾陈          官鬼戌土′    
朱雀          父母申金″    
青龙 妻财午火 兄弟亥水′ 世 
玄武          官鬼丑土″    
白虎          子孙卯木○    官鬼辰土″
```

分析：

- 忌神兄弟持世，飞克伏，不太好。
- 妻财月破，不好，虽然临日辰，但是引起兄弟暗动，克妻财，自然不好。用神能量比较弱。
- 此卦重点在于2处：应爻临月建冲破妻财，日辰冲兄弟暗动克财。还有一个，子孙独发化鬼。
- 兄弟持世，同时兄弟临应，世应都克了妻财，自己和别人共享一个蛋糕。男占感情，说明这个应爻就是情敌了。妻财月破，差。应爻兄弟临月建冲破用神妻财，感情低迷是这个应爻引起的。
- 个人经验，子孙独发生妻财，妻财月生扶才应好事。子孙独发化鬼，妻财也没有月生扶不太好。此卦，妻财月冲克，子孙发动，求财/向女人求爱，子孙化鬼，陷入绝望。

:::tip

这个既济的世应有点说法：

- 男占感情，这个应爻就是情敌了；
- 男占同性恋，这应爻是他的同性对象；
- 女平常占，这应爻是男方的朋友。（兄弟合官鬼）

:::





# 六神组合类



## 腾蛇

:::tip

腾蛇空亡，
其一，代表心慌，心神不宁
其二，代表虚假，欺骗，不实在，用假的来糊弄。

如：腾蛇妻财空亡，合了世爻/内卦，容易收到假钱。

:::

```
占题：韵达快递信息怎么回事
丁酉年 乙巳月 癸卯日 庚申时（空亡:辰巳）
风火家人 (巽宫)      泽火革 (坎宫)
白虎          兄弟卯木○    妻财未土″
腾蛇          子孙巳火′ 应 
勾陈          妻财未土×    父母亥水′
朱雀 官鬼酉金 父母亥水′    
青龙          妻财丑土″ 世 
玄武          兄弟卯木′    
```



分析：

- 世爻为自己，父母为快递，应爻代表自己对头的那方

- 应爻落空临腾蛇，带有虚假之象，对方不可信。
- 父母代表快递，月破，快递有点问题，下伏藏官鬼，代表盗贼，搞破坏，不怀好意的人，指定有点问题。

电话号码和信息是对的，订单号是假的，没有理睬。
