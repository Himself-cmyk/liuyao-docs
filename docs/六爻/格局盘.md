## 兄弟持世，邻爻兄弟发动

写成：`世爻 兄弟 安静，世爻邻爻 发动 扶世`

这个动爻就可以看作自己的同辈，同事，朋友。

```
丁酉年乙巳月戊申日庚申时
泽火革火雷噬嗑

```

结果：另外一个同事自己辞职了。



## 世爻安静，世爻替身发动

写成：`世爻 安静，动爻 扶世`

这个动爻也可以看成自己的同伴，同事。
